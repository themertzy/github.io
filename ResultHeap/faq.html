What is a static website?

A static website contains web pages with fixed content, meaning each page is coded in HTML and displays the same information to every visitor to your site. Static sites are used most by small businesses wwho are simply looking to display content.

What is a dynamic website?

Dynamic websites render pages in real-time. When someone accesses a dynamic web page, the server takes the content and "pastes" it onto the site, depending on the need.

Which type of website should I choose?

If you are a small business or individual simply looking to display some content then a static website is for you. If you are not well versed in coding yourself, then you should use a dynamic website as these give you more control as a beginner who wants to get your site off the ground.
